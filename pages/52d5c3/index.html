    Markdown 语法

    # 基础语法

    ​ 标题 : CTRL+ 1/2/3/4/5/6, 升级 CTRL + = ，降级CTRL + -

    文本前加井号（#），可以使用符号的数量与标题的级别相对应 ,

    ​ 段落 ： CTRL + O

    段落之间至少空一行

    ​ 换行 ：

    以至少两个空格结束一行，然后回车

    ​ 着重：

    • 粗体 ： 文本前后各加两个星号（*）CTRL + B
    • 斜体 : 文本前后各加一个星号 CTRL + I
    • 粗体加斜体 ： 文本前后各加三个星号

    ​ 块引用： CTRL + SHIFT + Q

    • 具有多个段落的块引用 ：在段落之前和空格行加一个 大于号（>）
    • 嵌套块引用 ：在要嵌套的段落之前加两个大于号
    • 具有其他元素的块引用 ：块引用可以包含其他Markdown格式的元素。并非所有元素都可以使用 例如： 在段落前加 > +

    ​ 列表 ：

    • 有序列表 : 在每一项前加数据和点符号，数字不必有序，但列表应以数字开头 CTRL + SHIFT + [

    • 无需列表 ：在每项前加破折号或者星号或者加号 ，缩进一个或者像项以创建嵌套列表 **CTRL + SHIFT + ] **

    • 在列表中加元素 ： 将该元素缩进四个空格或者一个制表符

    ​ 代码 ： CTRL + SHIFT + '

    将代码写在勾号中（`）

    ​ 代码块 ：

    在代码块的每一行缩进至少四个空格或者一个制表符

    ​ 水平线 ：

    在单独一行上使用三个或更多的星号（*）、破折号（-）、下划线（—）

    ​ 链接 ：CTRL + K

    将链接文本放在中括号（[ ]）,然后在后面加上括号（ （）） ，链接地址和链接描述放在括号里

    ​ 图片 ： CTRL + SHIFT + I

    先写一个感叹号加中括号（！[]） ，中括号中加代替文本，在后面加一个括号（），括号里写图片地址和图片描述，如果要向图片中加链接的话，请将 图片的Markdown括在中括号里

    # 扩展语法

    ​ 表格 ：在每列的下方至少使用三个连字符(---) , 并使用管道(|)分隔每列 CTRL + T

    • 对齐 ：在连字符的两侧或者左边、右边添加一个冒号(😉 ,使用列中的文本居中或者左、右对齐
    • 格式化表格中的文本 ：可以格式化表格中的文本。例如，您可以添加链接，代码（```仅在刻度线（）中显示单词或短语，而不能在代码块中添加）和强调。

    ​ 围栏代码块 ：CTRL + SHIFT + K

    在代码块的之前和之后的行上使用三个刻度线（'''）或者三个波浪号（~~~）

    ​ 删除线 ： ALT + SHIFT +5

    前后加上两个波浪线（~~）

    ​ 任务列表 ：

    在每一项前加破折号和中括号，破折号和括号之间要加一个空格，中括号里要加一个空格或者x，空格代表未完成，x代表已完成，括号与文字之间要加一个空格

    ​ 禁用自动URL链接 ：

    Markdown处理器会自动将URL转换为链接，如果不希望自动转换，则可以在URL前后加上刻度线（‘）
